Using Sass

  • First you write your code in an scss file
  • Then you run the compile script that you put into your package.json file. This will compile the sass code into whatever file you dictate
  • To run the compiler go to terminal and enter: npm run compile_sass
  • Open your website, as long as you have the css page linked that you compiled your code into you should be set.
